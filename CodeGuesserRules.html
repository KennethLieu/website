Code Guesser Rules:

The game will start off with the CPU generating a 3 digit code between 1 and 999. Your goal is to guess the correct code within 20 guesses. You are allowed to use a hint which uses up 10 guesses and reveals the code's hundreds digit. The game ends when you run out of guesses or you guess the correct code. After game ends, whoever wins most points wins the game, and if tie exists, tiebraker will be by random draw!
