UNO Classic Scoring:

Number cards (0-9) = face value points (0-9 points)
ACTION cards (REVERSE, SKIP, DRAW2) = 20 points
WILD cards (normal WILD, DRAW4WILD) = 50 points

2 player mode:
1st place score = 2nd place deck value
2nd place score = 0

4 player mode:
1st place score = 2nd + 3rd + 4th place deck values
2nd place score = 3rd + 4th place deck values
3rd place score = 4th place deck values
4th place score = 0
